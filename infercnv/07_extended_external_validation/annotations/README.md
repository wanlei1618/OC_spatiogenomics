# Original external scRNA annotations

This directory stores lightweight manifests and harmonized annotation tables for the external scRNA annotation upgrade.

Large source files are kept outside the repository under:

`D:/OC_spatiogenomics/infercnv/external_cell_annotations`

The harmonized tables preserve original labels in `cell_type_original`, map broad classes through `config/celltype_harmonization_map.csv`, and keep dataset roles from `config/external_scrna_dataset_registry.csv`.
