CNV + 表达联合分析生信方案

对象：integrated_oc；目标：inferCNV clone 与转录状态、EMT、Hypoxia、TF activity 的耦合分析

本文档整理了可直接围绕 Seurat 对象 integrated_oc 实现的分析方案。整体思路参考上传文献中“inferCNV 识别 CNV clone，并联合差异表达、EMT/hypoxia score 与 TF activity 构建克隆演化模型”的分析逻辑。

0. 总体分析框架

目标是将 inferCNV 得到的肿瘤 CNV clone 与表达状态、功能 signature、转录因子活性结合，构建“基因组拷贝数改变 → 转录状态改变 → 功能克隆演化模型”。

integrated_oc

├── CNV_clone

├── CNV_burden

├── chromosome-level CNV score

├── clone-specific DEG

├── EMT_score

├── Hypoxia_score

├── KRAS_score / LAPTM5 expression

├── TF activity

└── CNV-transcriptome coupling model

不同 CNV clone 是否对应不同表达程序？

哪个 clone 更 EMT-high、hypoxia-high、KRAS-active、LAPTM5-high？

哪些 TF 可能驱动这些 clone 的表型？

CNV burden 是否与 EMT/Hypoxia/KRAS 等恶性程序耦合？

15. 可写入论文/课题汇报的结论框架

首先，基于 inferCNV 推断的拷贝数变异模式，我们将 integrated_oc 中的肿瘤细胞划分为多个 CNV-defined subclones。

随后，通过差异表达分析发现，不同 CNV clone 具有显著不同的转录特征。

其中，CNV3/CNV4 等 high-CNV clone 表现出更高的 EMT score、Hypoxia score 和 KRAS signaling score。

进一步的 TF activity 分析显示，SNAI2、ATF3、HIF1A、STAT3、MYC 等转录因子在 high-CNV clone 中活性增强。

CNV burden 与 EMT/Hypoxia/KRAS score 的正相关提示，基因组拷贝数异常可能与肿瘤细胞侵袭性转录状态耦合。

因此，可构建 CNV-driven transcriptional evolution model：低 CNV epithelial-like clone 逐渐演化为 high-CNV EMT/hypoxia/KRAS-active clone，并可能进一步参与 M2 macrophage-associated tumor microenvironment remodeling。
